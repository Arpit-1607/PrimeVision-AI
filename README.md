# 🌸 Iris Flower Classification using Machine Learning

## 📌 Overview

This project demonstrates the implementation of a supervised machine learning pipeline for multi-class classification using the Iris dataset. The objective is to train and evaluate different classification algorithms capable of predicting flower species based on their physical measurements.

The project covers the complete machine learning workflow, including data loading, preprocessing, train-test splitting, model training, prediction, evaluation, and algorithm comparison.

---

## 🎯 Objectives

- Understand and explore a real-world dataset
- Apply supervised learning techniques
- Train and evaluate classification models
- Compare multiple machine learning algorithms
- Analyze model performance using accuracy metrics

---

## 📊 Dataset Information

**Dataset:** Iris Dataset

The dataset contains 150 samples of iris flowers belonging to three species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

### Features

| Feature | Description |
|----------|-------------|
| Sepal Length | Length of sepal (cm) |
| Sepal Width | Width of sepal (cm) |
| Petal Length | Length of petal (cm) |
| Petal Width | Width of petal (cm) |

### Target Variable

Flower Species

## 🤖 Machine Learning Algorithms

The following classification algorithms were implemented and evaluated:

### 1. Decision Tree Classifier
A tree-based model that makes predictions through a sequence of decision rules.

### 2. K-Nearest Neighbors (KNN)
A distance-based algorithm that classifies data based on neighboring samples.

### 3. Support Vector Machine (SVM)
A powerful classification algorithm that finds the optimal decision boundary between classes.
